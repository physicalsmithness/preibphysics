Setup / troubleshooting

If you see this panel, your Firebase config probably isn't set.

  1. Open config.js and paste your Firebase web app config.
  2. Enable Google sign-in in Firebase Authentication.
  3. Create Firestore and apply the security rules in firestore.rules.
  4. Deploy with Firebase Hosting (instructions in README.md).
